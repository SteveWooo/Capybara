(async, static) brocast(args) → {GenericResponse}

发送窗口间的广播信息。注意，窗口未完全启动的时候是没法收到信息的
Parameters:
Name Type Description
args Object 广播接口参数对象
Properties
Name Type Description
content Object 广播给其他窗口的内容
Since:
  • release-0.0.2
Returns:
通用返回
Type
GenericResponse
Example
const res = await window.kt.window.brocast({
    content: {
        hello: 'every windows' 
    }
})
// res.status === 2000 -> √

// 接收端：
kt.on.windowBrocast((e, data) => {
    console.log(data.content) // => { hello: 'every window }
    console.log(data.from) // => 发起广播的窗口信息
})

(async, static) close() → {GenericResponse}

关闭当前窗口, 或通过窗口id，关闭其他窗口
ps: 关闭窗口后，线程、DBSet的window作用域内的数据都会被清理干净
因此类似便利贴之类的应用，应谨慎使用该接口
Parameters:
Name Type Description
args.__wid String 窗口ID，为空则关闭当前窗口
Since:
  • release-0.0.2
Returns:
- 通用返回
Type
GenericResponse
Example
await kt.window.close()

设置窗口的位置、大小。

Name Type Attributes Default Description
