Name Type Description

})

  • Object

Name Type Attributes Default Description
__wid string 窗口唯一标识
x number <optional>

y number <optional>
 100 窗口在屏幕上的y坐标
width number <optional>
 380 窗口宽
height number <optional>
 220 窗口高
type string <optional>
 toolbar 对应electron WebBrowser的type
frame boolean <optional>
 true 应用是否有边框
resizable boolean <optional>
 true 应用是否有边框
transparent boolean <optional>
 false 背景是否透明
hasShadow boolean <optional>
 false 设置窗口是否有阴影
alwaysOnTop boolean <optional>
 true 窗口是否永远在别的窗口的上面
kt_devTools boolean <optional>
 false 是否启动控制台
kt_isolateSessionLevel String <optional>
 "app" 窗口使用的session隔离程度。可选：window, app, global
kt_recordRectInfo boolean <optional>
 true 是否记录窗口的大小、位置信息
kt_showWhenStartup boolean <optional>
 true 是否初次启动就显示窗口。（release-0.0.2）
kt_extraElectronBrowserWindowConfig Object <optional>
 {} 透传窗口配置给electron WebBrowser (release-0.0.2 及以上版本支持)
