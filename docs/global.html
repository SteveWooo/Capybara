 100 窗口在屏幕上的y坐标

 380 窗口宽

 220 窗口高

 toolbar 对应electron WebBrowser的type
frame boolean <optional>
 true 应用是否有边框
resizable boolean <optional>
 true 应用是否有边框
transparent boolean <optional>
 false 背景是否透明
hasShadow boolean <optional>
 false 设置窗口是否有阴影
alwaysOnTop boolean <optional>
 true 窗口是否永远在别的窗口的上面
fm_devTools boolean <optional>
 false 是否启动控制台
fm_isolateSessionLevel String <optional>
 "app" 窗口使用的session隔离程度。可选：window, app, global
fm_webSecurity boolean <optional>
 true 是否开启浏览器安全策略。关闭该参数前，请充分了解浏览器 CSP 的作用⚠
fm_recordRectInfo boolean <optional>
 true 是否记录窗口的大小、位置信息
fm_closeWarn boolean <optional>
 true 关闭警告提示。打开该开关的挂件，关闭的时候会弹窗提醒。建议像便利贴之类的依赖窗口数据库的应用设置为true。
fm_showWhenStartup boolean <optional>
 true 是否初次启动就显示窗口。（release-0.0.2）
fm_extraElectronBrowserWindowConfig Object <optional>
 {} 透传窗口配置给electron WebBrowser (release-0.0.2 及以上版本支持)
