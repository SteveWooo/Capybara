# async static batch() → {GenericResponse}

从 levelDB 批量操作。批量操作时，只能进行 put 和 del 操作。
Parameters:
Name Type Description
args.key string 数据键
args.batchData * 批量操作内容
args.options Obejct 其他 batch 参数选项 ，详细参考：https://github.com/Level/level#chainedbatch--dbbatch
args.sublevel string 子数据域，为空代表项目根域
Since:
  • release-0.0.2
- 通用返回
GenericResponse
Example
await fm.levelDB.batch({
    batchData: [
        { type: 'put', key: 'a', value: 'foo' },
        { type: 'put', key: 'b', value: 123, valueEncoding: 'json' }
    ]
})

# async static clear() → {GenericResponse}

从 levelDB 清空数据
Parameters:
Name Type Description
args.sublevel string 子数据域，为空代表项目根域
args.options string 对应 https://github.com/Level/level#dbclearoptions-callback
Since:
  • release-0.0.2
res - 通用返回
GenericResponse
Example
// 清空根域下数据
await fm.levelDB.clear()
// -> {status: 2000, result: []} 会将所有子域下的数据都清空

// 清空子域下数据
await fm.levelDB.clear({sublevel: 'sl'})
// -> 清空了

// 带条件清空数据
await fm.levelDB.clear({ options: { limit: 1 } }})
// -> 只清空了一条数据

# async static get() → {GenericResponse|string}

从 levelDB 读数据
Parameters:
Name Type Description
args.key string 数据键
args.sublevel string 子数据域，为空代表项目根域
Since:
  • release-0.0.2
res - 通用返回
GenericResponse
res.result - 返回的数值
string
Example
// 根域下读写
await fm.levelDB.put({key: 'hello', value: 'world'})
await fm.levelDB.get({key: 'hello'}) // -> {status: 2000, result: 'world'}

// 子域下读写数据：
await fm.levelDB.put({key: 'hello', value: 'world', sublevel: 'sl'})
await fm.levelDB.get({key: 'hello', sublevel: 'sl'}) // -> {status: 2000, result: 'world'}

# async static iterator() → {GenericResponse|string}

从 levelDB 遍历数据
Parameters:
Name Type Description
args.key string 数据键
args.sublevel string 子数据域，为空代表项目根域
args.options string 对应 https://github.com/Level/level#iterator--dbiteratoroptions
Since:
  • release-0.0.2
res - 通用返回
GenericResponse
res.result - 返回的数值
string
Example
// 根域下遍历
await fm.levelDB.iterator()
// -> '{"status":2000,"result":[{"key":"!sl!hello","value":"world"},{"key":"hello","value":"world"}]}'

// 子域下遍历
await fm.levelDB.iterator({sublevel: 'sl'})
// -> '{"status":2000,"result":[{"key":"hello","value":"world"}]}'

// 带条件遍历
await fm.levelDB.iterator({sublevel: 'sl', options: { limit: 1 }})
// -> 只返回一条数据

# async static put() → {GenericResponse}

从 levelDB 写数据
Parameters:
Name Type Description
args.key string 数据键
args.value * 数据值
args.sublevel string 子数据域，为空代表项目根域
Since:
  • release-0.0.2
- 通用返回
GenericResponse
Example
// 根域下写

await fm.levelDB.get({key: 'hello', sublevel: 'sl'}) // -> {status: 2000, result: 'world'}
