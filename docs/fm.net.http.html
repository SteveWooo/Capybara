Methods

# static httpRequest() → {GenericResponse|Number|String|Object|String}

发起一个http请求，会跨域的哟
Parameters:
Name Type Description
args.url string 请求url
args.method string 请求方法。默认GET
args.headers Object 额外请求头
args.body String 请求体。请注意是String类型字段
args.extraRequestParam Object 其他axios请求参数字段，会以最高优先级附带覆盖到axios请求参数上，详情参考：https://axios-http.com/docs/req_config
Since:
  • release-0.0.2
- 通用返回
GenericResponse
res.httpStatusCode - http状态码
Number
res.data - 文本形式的返回
String
res.json - 如果请求结果是json，则返回json结构的数据
Object
res.message - 请求错误信息
String
Example
// GET请求
const res = await kt.net.http.request({
    url: 'https://www.yourapi.com/path',
    method: 'GET',
})

// POST请求
const res = await kt.net.http.request({
    url: 'https://www.yourapi.com/path'，
    method: 'POST',
    headers: {
        'Content-Type': 'Application/json'
    },
    body: JSON.stringify({ 'Hello': 'KTool' }),
    extraRequestParam: {
        timeout: 5000
    }
})
