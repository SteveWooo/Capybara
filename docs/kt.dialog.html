(async, static) showMessageBox(options) → {GenericResponse|*}

打开一个MessageBox提示框
Parameters:
Name Type Description
options Object 对话框入参。参数详情请参考 https://www.electronjs.org/zh/docs/latest/api/dialog#dialogshowmessageboxsync options
Since:
  • release-0.0.2
Returns:
  • - res 通用返回
    Type
    GenericResponse
  • - res.result 透传 electron showMessageBoxSync 结果
    Type
    *
Example
let response = await kt.dialog.showMessageBox({
    message: '你确定吗', 
    title: '警告', 
    type: 'warning', 
    buttons: ['ok', 'cancel']
})
console.log(response.result) // => 按了OK的话就会返回 0 了。
