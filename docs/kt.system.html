(async, static) getInfo() → {GenericResponse|Object}

获取系统信息，具体可参考：https://systeminformation.io/general.html 的get函数用法
感谢 https://systeminformation.io 这个优秀的项目，大家快去 Buy him a coffee。
Parameters:
Name Type Description
args.valueObject Object 参考systeminformation的get函数入参
Since:
  • release-0.0.2
Returns:
  • res - 通用返回
    Type
    GenericResponse
  • res.result - 透传返回
    Type
    Object
Example
const systemInfo = await kt.system.getInfo({
   valueObject: {
     currentLoad: 'currentLoad',
     mem: 'used total',
     networkInterfaces: 'ip4 iface',
   }
})

console.log(systemInfo.result.networkInterfaces.ip4) => ipv4 address
